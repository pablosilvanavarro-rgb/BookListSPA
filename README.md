📚 BookList - Editorial Nova
SPA (Single Page Application) desarrollada con Vue.js 3 para la gestión de catálogos literarios. El proyecto permite registrar libros, filtrarlos en tiempo real y visualizar detalles mediante rutas dinámicas.

✨ Características
Registro Reactivo: Formulario con validación y vinculación bidireccional (v-model).

Catálogo Dinámico: Renderizado de listas con v-for y filtrado inteligente mediante propiedades computadas.

Rutas Dinámicas: Navegación fluida con Vue Router para fichas de detalle.

Persistencia: Almacenamiento local mediante LocalStorage.

Diseño Modular: Estilos encapsulados con Scoped CSS.

🛠️ Tecnicas usadas
Vue.js 3 (Composition API / Options API)

Vue Router

JavaScript ES6+

LocalStorage API
